## 機能

緊急地震速報や地震情報（震度速報、震源情報、遠地地震）をリアルタイムで受信し、チャットに情報の詳細が送信されます。

## 謝礼

### 地震情報API
- P2PQuake JSON API v2

### 緊急地震速報API
- Wolfx API
